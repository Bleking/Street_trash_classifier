
Street trash - v3 Trash dataset
==============================

This dataset was exported via roboflow.ai on June 24, 2022 at 3:44 AM GMT

It includes 679 images.
Litters are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Auto-orientation of pixel data (with EXIF-orientation stripping)

No image augmentation techniques were applied.
